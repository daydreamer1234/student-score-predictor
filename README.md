Student Score Predictor

Project Overview

This project predicts student marks based on study hours using Linear Regression.

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

Dataset

The dataset contains:

- Study Hours
- Marks Obtained

How It Works

The model learns the relationship between study hours and marks, then predicts marks for new input values.

Example Prediction

If study hours = 7

Predicted Marks = 75 (example)
